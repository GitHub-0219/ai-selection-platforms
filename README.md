---

## ✨ 功能特性

| 模块 | 功能说明 |
|------|----------|
| 🎯 AI 选品 | 智能推荐算法，精准匹配用户偏好 |
| 📈 趋势分析 | 实时追踪时尚/批发行业趋势 |
| 🎨 设计中心 | 可视化选品设计与搭配方案 |
| 📊 数据看板 | 多维度数据统计与可视化图表 |
| 👤 用户中心 | 个人信息、收藏夹、历史记录管理 |
| 📱 响应式 | 完美适配桌面端、平板、手机端 |

## 🚀 快速开始

### 方式一：直接打开

```bash
# 克隆仓库
git clone https://github.com/GitHub-0219/ai-selection-platforms.git
cd ai-selection-platforms

# 直接用浏览器打开
open ai-fashion/index.html      # 时尚选品平台
open ai-wholesale/index.html    # 批发选品平台
```

### 方式二：本地服务器

```bash
# 使用 Python 启动本地服务
cd ai-selection-platforms
python3 -m http.server 8080

# 访问 http://localhost:8080/ai-fashion/
# 访问 http://localhost:8080/ai-wholesale/
```
